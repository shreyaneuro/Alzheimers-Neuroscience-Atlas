
# Alzheimer's Disease – A Comprehensive Neuroscience Atlas

## Overview

This repository presents an independent neuroscience project exploring Alzheimer's disease from normal brain anatomy to molecular mechanisms, disease progression, biomarkers, neuroimaging, therapeutic strategies, and future research.

The project integrates current scientific literature and provides a comparative evaluation of major diagnostic biomarkers including MRI, PET imaging, cerebrospinal fluid biomarkers, and emerging blood-based biomarkers.

---

## Objectives

- Explain healthy brain anatomy
- Describe Alzheimer's disease pathology
- Compare major biomarkers
- Review MRI and PET imaging
- Discuss current treatments
- Explore future research directions
