# Alzheimer's Disease – A Comprehensive Neuroscience Atlas

## Overview

This repository contains an independent literature-based neuroscience project exploring Alzheimer's disease from healthy brain anatomy to molecular mechanisms, biomarkers, neuroimaging, therapeutic strategies, and future research.

## Project Contents

- Healthy Brain Anatomy
- Brain Regions Affected in Alzheimer's Disease
- Molecular Mechanisms
- Disease Progression
- MRI and PET Imaging
- Biomarker Comparison
- Current and Emerging Treatments
- Future Research Directions
